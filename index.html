🚀 Twenty CRM on Coolify

Self-host Twenty CRM with one-click Coolify deployment

What's included

✅ Coolify-native Docker Compose configuration
✅ Automatic HTTPS via Traefik + Let's Encrypt
✅ PostgreSQL 16 + Redis 7 with health checks
✅ Background worker for async jobs
✅ Persistent storage for uploads & database

Stack

Twenty CRM PostgreSQL 16 Redis 7 Traefik Coolify

Quick Start

1 Create a new Docker Compose Empty resource in Coolify
2 Copy the contents of twenty-docker-compose.yaml
3 Configure your domain in Coolify and enable HTTPS
4 Deploy! Coolify handles certificates automatically
